# 🍽️ MealMitra

MealMitra is a full-stack food ordering and delivery platform that connects customers with home-based cooks and small food businesses.

The platform allows users to browse meals, place orders, and manage food listings with secure authentication.

--- 

## 🚀 Features

• User registration and login  
• Secure authentication using JWT  
• Browse meals from different cooks  
• Place and manage food orders  
• Integrated payment gateway using C# .NET  
• Role-based access control for users and vendors  
• RESTful API backend for food ordering operations

## 🏗 System Architecture

MealMitra follows a full-stack architecture:

• React frontend for user interface  
• Spring Boot backend for REST APIs and business logic  
• MySQL database for storing users, meals and order data  
• C# .NET service for handling payment gateway integration

## ⚙️ How to Run the Project

1. Clone the repository

git clone https://github.com/kushwahasneha893/MealMitra.git

2. Start the backend

3. Configure MySQL database

Update database credentials in:

application.properties

4. Start the frontend
